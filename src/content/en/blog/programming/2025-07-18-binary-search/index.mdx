What is Binary search?

An algorithm used to search for a specific element from sorted elements. How does it search:

1. Find the middle number.
2. If our searching element is smaller than the middle number, then search among the numbers smaller than the middle number.
3. If our searching element is larger than the middle number, then search among the numbers larger than the middle number.

![Binary search](https://media.geeksforgeeks.org/wp-content/uploads/20240506155201/binnary-search-.webp)

```
bool binarySearch(int* arr,int left,int right ,int searchingNumber){
int mid;
while(left<=right){
    mid=left+(right-left)/2;
    if(arr[mid]==searchingNumber){
        return true;
    }
    if(arr[left]<arr[mid])
    {
        right=mid-1;
    }
    else {
        left=mid+1;
    }
}
return false;
}
```
